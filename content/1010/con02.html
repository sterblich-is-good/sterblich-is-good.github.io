安装Git 配置 Git 用户信息 生成 SSH Key（推荐） 克隆仓库到本地 本地修改并推送到 GitHub 常用命令总结

这是页面主标题

安装Git

Windows：Git for Windows

Mac: 

brew install git

Linux: 

sudo apt-get install git

配置 Git 用户信息

输入：


          git config --global user.name "你的GitHub用户名"
          git config --global user.email "你的GitHub邮箱"

生成 SSH Key（推荐）

这样以后每次推送不用输入账号密码。

1. 生成 SSH key：


          ssh-keygen -t rsa -b 4096 -C "你的GitHub邮箱"

2. 查看公钥：


          cat ~/.ssh/id_rsa.pub

复制输出内容

3. 打开 GitHub SSH keys 设置 ，点击 New SSH key，把复制的公钥粘贴进去并保存。

测试连接：


          ssh -T git@github.com

如果提示 Hi 用户名! You've successfully authenticated 就成功了。

克隆仓库到本地

在你想放项目的文件夹运行（如果是windows的话在文件夹下可以右击选择Open Git Bash here）：


          git clone git@github.com:用户名/仓库名.git

本地修改并推送到 GitHub

1. 进入项目目录：


          cd 仓库名

2. 新建或修改文件。

3. 添加到暂存区：


          git add .

4. 提交：


          git commit -m "你的提交说明"

5. 推送到Github：


          git push origin main
注意：有些仓库主分支叫 master，那就改成 git push origin master。

常用命令总结


          git pull origin main      # 拉取最新代码
          git status                # 查看文件状态
          git branch -a             # 查看所有分支
          git checkout 分支名        # 切换分支
          git push origin 分支名     # 推送到指定分支
